☁️ Synchronisation multi-appareils

Colle ici l'URL de ta base Firebase pour partager tes données entre ton téléphone, ta tablette et ton PC.

Comment créer ta base gratuite :
  1. Va sur console.firebase.google.com
  2. Crée un projet (nom au choix)
  3. Dans le menu gauche → Realtime Database → Créer une base
  4. Choisis Mode test (30 jours gratuit, modifiable ensuite)
  5. Copie l'URL affichée (ex: https://mon-projet-default-rtdb.firebaseio.com)
  6. Colle-la ici — même URL sur tous tes appareils !
